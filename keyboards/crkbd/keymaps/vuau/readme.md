# Atreus to Corne (crkbd) Keymap Port

This keymap is a port from the Keyboardio Atreus keymap. Since Atreus has 44 keys and Corne has 42 keys, the layout has been adapted by moving the inner keys of Atreus to the outer columns of the Corne.

### Mouse Key Config:
Constant mouse speed without acceleration is enabled via `config.h`.

### Mapping Logic:
- The 5 main columns of Atreus are mapped to the 5 inner columns of Corne.
- Atreus inner column keys (e.g., `KC_VOLD`, `KC_VOLU`) are moved to the outer bottom row of Corne.
- Outer modifiers/extra keys on Atreus are moved to the top and middle outer columns of Corne.
- The 3 main thumb keys of Atreus are mapped directly to the 3 thumb keys of Corne.
